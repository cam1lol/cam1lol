Desarrollo aplicaciones web full stack con foco en arquitectura limpia, interfaces que se sienten rápidas y experiencias 3D que corren directo en el navegador. Más de dos años construyendo plataformas reales — desde sistemas electorales con Angular Signals hasta pipelines conversacionales con IA.

Paralelamente fundé **Velamentum**, una marca de ropa donde cada prenda es un modelo 3D interactivo: diseñada en Blender, exportada en GLB y renderizada en el browser con Three.js. Sin apps, sin plugins.

---

## Stack

| Área | Tecnologías |
|---|---|
| Frontend | Angular v14–v20+ · Vue.js · TypeScript · Three.js |
| Backend | Java + Spring Boot · Node.js + Express · Python + FastAPI |
| Bases de datos | PostgreSQL · MySQL · SQL Server · MongoDB |
| Cloud & DevOps | AWS (AppSync · Lambda · S3) · Azure DevOps · Docker · CI/CD |
| 3D & Visual | Blender 5.1 · GLB/GLTF · CSS avanzado |
| IA & Bots | Langchain · GPT API · NLP · Webhooks multicanal |
